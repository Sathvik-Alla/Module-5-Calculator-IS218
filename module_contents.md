# Module 5 - Enhanced Calculator Application

## Overview
This project implements a command-line calculator using advanced Python design patterns and pandas for data management.

---

## Features
- REPL (Read-Eval-Print Loop)
- Arithmetic operations (add, subtract, multiply, divide, power, root)
- Factory Pattern (operation creation)
- Strategy Pattern (operation execution)
- Observer Pattern (history tracking)
- Memento Pattern (undo/redo)
- pandas-based CSV history storage
- Configuration via dotenv
- Robust exception handling
