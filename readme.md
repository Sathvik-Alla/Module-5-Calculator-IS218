# 🧮 Enhanced Calculator Application (Module 5)

## Overview
This project is a Python-based CLI calculator that demonstrates advanced design patterns, pandas data management, and automated testing.

---

## Features
- REPL interface
- 6 arithmetic operations
- pandas CSV history tracking
- undo/redo system
- configuration via .env
- full pytest test suite
- GitHub Actions CI pipeline
